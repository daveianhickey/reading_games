# Essential Phonics List
Based on the provided ELS (Essential Letters and Sounds) charts, these are the key letter combinations the AI should focus on.

## Digraphs (Two letters, one sound)
    • Consonant Digraphs: ck, ss, ff, ll, zz, qu, ch, sh, th, ng, nk
    • Vowel Digraphs: ai, ee, oa, oo (long and short), ar, ur, or, ow, oi, er, ure

## Trigraphs (Three letters, one sound)
    • Vowel Trigraphs: igh, ear, air
